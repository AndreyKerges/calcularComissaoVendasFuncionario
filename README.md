# calcularComissaoVendasFuncionario
Este exercício lê o nome do funcionário e a quantidade de produtos vendidos. A comissão é calculada com base nas seguintes regras: R$ 1,00 por produto vendido até 250 produtos; R$ 2,50 por produto para vendas entre 251 e 500 produtos; e R$ 3,00 por produto para vendas acima de 500 produtos.
